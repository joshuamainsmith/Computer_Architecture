Create text file named import.txt

Format should be:

Name of Dept.
Phone #

Example:

Sociology Dept
(603) 543-9344

When clicking on the executable, a file named export.txt will be generated.

The output will be in the form of"

<div><strong>Name of Dept</strong></div>
<div>Phone #</div>
<br>

Example:

<div><strong>Sociology Dept</strong></div>
<div>(603) 543-9344</div>
<br>
